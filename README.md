# HempTemps App

Project created with

- Vanilla React v18
- Tailwind
- JWT Authentication

### Instructions to use ESLint:

1. Get the ESLint extension

<img width="1997" alt="image" src="https://github.com/Hemptemps-com/hemptemps-app/assets/96852364/a6230bbf-cc6b-4540-ab4f-a3262ba97c2c">

2. Open Settings (UI) page in VS Code via CommandPalette (SHIFT+COMMAND+P)

<img width="1341" alt="image" src="https://github.com/Hemptemps-com/hemptemps-app/assets/96852364/ece37a19-b2aa-486b-a1a2-4d9297042db5">

3. Search for 'eslint' and in the options below, ENABLE 'Eslint > Experimental: Use Flat Config'

<img width="1498" alt="image" src="https://github.com/Hemptemps-com/hemptemps-app/assets/96852364/19829fc3-f4a5-40b5-af1a-b88574ec363c">

4. Close settings tab. Enjoy real time syntax highlighting for any errors and warnings as you code. Cheers!
